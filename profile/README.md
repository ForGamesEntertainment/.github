# 🎮 Welcome to ForGames Entertainment

## 🗡️ Malwarian

### About the Game

**Malwarian** is an exciting roguelite, bullet heaven, action game. Kill enemies, increase strength, develop character and more.

### 🎯 Game Features

- **Roguelite Gameplay**: Experience dynamic, procedurally generated challenges with permanent character progression
- **Combat System**: Engage in fast-paced battles against diverse enemies
- **Character Development**: Build your character with unique augments and abilities
- **Augment System**: Choose powerful augments when leveling up to customize your playstyle
- **Progressive Difficulty**: Each run presents new challenges and opportunities for growth
